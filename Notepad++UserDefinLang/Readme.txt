
How to Use This UDL File (User Defile Lang colorset for Vim Dark Blue theme)

Save the file as LaTeX-HebrewTemplate.udl.xml in the folder 
C:\Users\gal-t\AppData\Roaming\Notepad++\userDefineLangs


Import into Notepad++:

	Open Notepad++
	Go to Language → User Defined Language → Define your language...
	Click Import...
	Select the LaTeX-HebrewTemplate.udl.xml file
	Restart Notepad++


Apply to your files:

	Open a .tex, .cls, , bib or .sty file
	Go to Language → YoramLaTeX
